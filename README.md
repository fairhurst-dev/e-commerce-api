# e-commerce-api

## prerequisites ⬅️

You will need:

- an AWS account
- a [serverless](https://www.serverless.com/) account
- a [stripe](https://stripe.com/) account

## setup 🔩

`git clone https://github.com/fairhurst-dev/e-commerce-api.git`

`cp .env.example .env`

`npm i serverless -g`

`serverless login`

## install 📦

`npm install`

## test 🧑‍🔬

`npm run test`

## deploy 🚀

`sls deploy`
